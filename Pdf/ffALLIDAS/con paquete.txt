El problema al ejecutar desde src:
Cuando ejecutas java P1.Lenguaje desde src:

Java busca: ./P1/Lenguaje.class

Pero tu archivo está en: ./P1/Lenguaje.class ← ¡Sí está ahí!

¿Entonces por qué falla? Porque Java interpreta P1.Lenguaje como:

Paquete: P1

Clase: Lenguaje

Ubicación esperada: P1/Lenguaje.class relativo al classpath
